
# Classes de veiculos

Projeto realizado para entrega em aula do curso de Análise e Desenvolvimento de Sistemas ULBRA.

Criadas três classes de objetos em Java com IntelliJ para demonstrar a herança de classes, neste programa é possível criar objetos de Carro e Moto, como dados importantes como Marca, modelo e ano, além disso existe uma função que possibilita a visualização dos objetos.

## Executar
Basta executar o arquivo com o nome Main.
É possível criar outros objetos seguindo o modelo já presente no código. Não esqueça de chamar a função informacoesVeiculo() para visualizar seu carro/moto.
